Frequently asked questions

What's the difference between a website and a web app?

A website is mostly read-only. A web app has user accounts, live data, forms that do things, and logic behind the scenes.

Can it connect to my existing software?

Most common SaaS tools have APIs. Katie will map the integration surface in the scoping call.

Who hosts it?

Your choice. Katie can deploy to your infrastructure or set up managed cloud hosting with transparent costs.

Do I need a database?

Almost certainly yes. Katie selects the right one (SQL vs NoSQL) based on your data shape and query patterns.
