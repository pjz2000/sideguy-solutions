# SideGuy Page Upgrade Engine

Purpose:

Improve existing pages across the SideGuy platform.

Large knowledge sites gain authority by continually improving existing pages.

This engine upgrades pages by adding:

- stronger introductions
- examples
- FAQ schema
- better internal linking

--------------------------------------------------

UPGRADE RULES

All upgrades must be:

append-only  
safe for SEO  
non-destructive  

Never remove existing content.

--------------------------------------------------

UPGRADE TYPES

Intro Upgrade

Ensure every page begins with a clear explanation.

Example format:

What this topic means for operators  
Why it matters  
How SideGuy helps explain it  

--------------------------------------------------

Example Section

Add a short real-world example where helpful.

--------------------------------------------------

FAQ Schema

Add common questions such as:

What does this mean?  
How does this work?  
Why does this matter for businesses?  

--------------------------------------------------

Internal Linking

Ensure each page links to:

homepage  
command center  
relevant knowledge hub  
two related pages
