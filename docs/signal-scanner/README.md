# SideGuy Signal Scanner

Purpose:
Track new technologies and signals that may become important.

SideGuy stays ahead of trends by identifying signals early
and creating reserve topics before they become mainstream.

Workflow:

Signal appears
→ log the signal
→ classify technology
→ create reserve topic
→ publish when demand grows

Signals may come from:

technology research
startup activity
scientific breakthroughs
hardware development
energy systems
automation advances

SideGuy does not chase hype.

It tracks real infrastructure changes.
