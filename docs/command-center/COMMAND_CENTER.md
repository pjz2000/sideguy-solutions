# SideGuy Command Center

Purpose:

Run the full SideGuy intelligence system in one command.

This script executes:

Signal Engine - future topic detection
Flywheel Engine - authority analysis
Priority Engine - page upgrade detection
Site Metrics - page counts and link density

Command:

bash tools/intelligence/sideguy-command-center.sh .

Output:

logs/command-center/command-center-report.txt

Workflow:

Run the command center scan.

Review the report.

Build 1-3 new pages.

Improve weak pages.

Expand clusters.

Repeat daily.

This creates continuous authority growth for the site.
