## Virality Distortion Sensor

When ingesting screenshots, tweets, videos, or "BREAKING" claims:

### Check 1: Calendar context
- Is it April 1?
- Is there major event bait?
- Is the timing suspicious?

### Check 2: Tone mismatch
- Does the quote sound unlike the company?
- Is the wording too perfect?
- Is the claim emotionally optimized?

### Check 3: Rebrand absurdity
- "OpenClaude"
- "GPT Infinity"
- "Google buys the internet"
- obvious meme names

### Check 4: Source verification
- official newsroom
- company blog
- exec account
- repeat credible sources

### Check 5: Market reality
Would this actually happen?
- If yes → build page
- If maybe → hold
- If meme → convert to humor asset
