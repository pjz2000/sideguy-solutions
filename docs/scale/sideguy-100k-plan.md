# SideGuy 100K Expansion Plan

Goal:

Grow SideGuy to 100,000 pages cleanly.

This is not a random page blast.
This is a structured expansion system.

--------------------------------------------------

CORE RULES

1. append-only
2. no deleting content
3. no giant uncontrolled generation
4. build in batches
5. every page must belong to a system

--------------------------------------------------

VALID PAGE TYPES

- problem pages
- cost pages
- comparison pages
- how-to pages
- local pages
- future infrastructure pages
- glossary / explainer pages

--------------------------------------------------

REQUIRED CONNECTIONS

Every new page must connect to:

- homepage
- command-center-v2.html
- at least one knowledge hub
- at least two related pages

--------------------------------------------------

QUALITY FLOOR

Every page should include:

- title
- meta description
- canonical
- H1
- intro paragraph
- at least 3 structured sections
- Text PJ orb
- Back to Home link

773-544-1231

--------------------------------------------------

SCALING METHOD

Use batches of 250 to 1,000 pages.

For each batch:

1. generate pages
2. validate slugs
3. validate required metadata
4. add to sitemap
5. add to index
6. add hub links
7. run health check
8. commit

Never generate 50,000 pages in one shot.

--------------------------------------------------

BATCH TYPES

1. Triangle clusters
   - how-to
   - comparison
   - cost

2. Problem clusters
   - why
   - how
   - what
   - cost

3. Local overlays
   - topic + city
   - topic + san-diego
   - topic + north-county
   - topic + carlsbad
   - topic + escondido

4. Industry overlays
   - contractors
   - law firms
   - solar
   - HVAC
   - medical
   - e-commerce
   - restaurants

--------------------------------------------------

TARGET ARCHITECTURE

100,000 pages should still feel like:

- one connected knowledge platform
- strong hubs
- clean internal linking
- premium human-readable UX

--------------------------------------------------

MISSION

Scale SideGuy like a platform, not a spam factory.
