# SideGuy Million-Dollar Site Standard

Purpose:
Ensure every update to the site makes SideGuy feel like a premium operator platform.

SideGuy should always look closer to a **$1M startup site** than a generic SEO site.

This rule applies to:
- new pages
- existing page upgrades
- homepage changes
- UI improvements
- content additions

--------------------------------------------------

CORE BRAND FEEL

SideGuy should feel:

calm  
premium  
coastal / ocean-light  
futuristic but human  
clear and trustworthy  
operator-focused  

Avoid:

generic templates  
flat black blocks  
ugly spacing  
giant text walls  
cheap looking sections  
inconsistent cards

--------------------------------------------------

DESIGN LANGUAGE

Preferred aesthetic:

ocean-light  
Solana-beach energy  
soft gradients  
glass / soft cards  
clean typography  
consistent shadows  
consistent border radius

Spacing should feel intentional and premium.

Cards should feel designed, not auto-generated.

--------------------------------------------------

PAGE STRUCTURE RULE

Every page should follow a strong structure:

1. Hero section
2. Clear explanation
3. Structured sections
4. Visual blocks or cards
5. Internal links
6. Calm CTA with Text PJ orb

calm
