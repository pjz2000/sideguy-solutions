# SideGuy Product-Level UI System

Goal:
Upgrade SideGuy from a typical SEO site into a premium product-style experience.

The site should feel closer to:

Stripe
Linear
Vercel
Notion

But with the SideGuy tone:
calm
human
clarity-first
coastal / light ocean aesthetic


Core principles:

1. Visual hierarchy
Clear spacing between sections.
Hero must dominate the first screen.

2. Premium cards
Soft shadows
Rounded corners
Subtle gradients
Consistent spacing

3. Minimal copy
Break long paragraphs into blocks.

4. Motion
Subtle hover effects.
No chaotic animation.

5. Operator aesthetic
Feels like a calm command center.
