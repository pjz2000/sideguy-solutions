# SideGuy Impression Optimization

Goal:
Improve pages already getting Google impressions.

Strategy:

1. Improve titles
2. Improve meta descriptions
3. Add internal links
4. Add FAQ blocks
5. Ensure Text PJ CTA present

Why it works:

Google already tests these pages.

Better CTR + stronger signals -> higher rankings.

## Usage

1. Export Search Console data to Table.csv (Page, Clicks, Impressions, CTR, Position)
2. Run: python3 tools/seo-boost/impression_boost.py
3. Review docs/seo-boost/impression-pages.md
4. For each page with high impressions / low CTR, update title and meta manually
