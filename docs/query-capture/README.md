# SideGuy Query Capture Engine

1) In Google Search Console:
   Performance → Search Results → Queries → Export → CSV

2) Save the file as:
   data/query-capture/gsc_queries.csv

3) Run:
   python3 tools/query-capture/query_capture.py

Outputs:
- docs/query-capture/page-ideas.txt
- docs/query-capture/page-ideas.csv
- docs/query-capture/top-clusters.txt
