# SideGuy Sensor Grid v1
Updated: 2026-04-01

## Mission
Turn the internet into a live sensor network that detects:
- demand spikes
- local confusion
- vendor frustration
- review pain
- trend emergence
- price volatility
- infrastructure anomalies

Then convert those signals into:
1. clarity pages
2. calculators
3. local trust pages
4. meme content
5. human escalation paths
6. vendor routing
7. future product opportunities

---

## Core Sensors

### 1) GSC Intent Sensor
Use impressions + CTR + new query emergence.

**Money use:**
- spawn child pages
- improve pages with live impressions
- FAQ expansion
- calculator insertion
- local variants

### 2) Reddit Confusion Sensor
Watch:
- repeated complaint phrasing
- "who do I call"
- local service failures
- AI fear / future confusion
- relocation pain

### 3) Review Pain Sensor
Track:
- HVAC bad reviews
- delayed vendors
- billing/payment frustration
- "nobody called me back"
- refund problems

### 4) YouTube Trend Sensor
Turn creator trend topics into:
- future service pages
- explainers
- meme responses
- local versions

### 5) Prediction Signal Sensor
Track:
- sports pricing inefficiencies
- regional economy moves
- weather-driven chaos
- travel delays
- event-driven demand spikes

### 6) Local Trust Sensor
Create North County trust references:
- Encinitas coffee shops
- Solana Beach routines
- Del Mar / airport routes
- local bars and stores
- neighborhood-specific decision pages

### 7) Meme Sensor
Convert internet frustration into:
- relatable humor
- "this is why SideGuy exists"
- shareable image ideas
- human trust moments
