# SideGuy Long-Tail Cluster Engine

Purpose:

Generate structured clusters of related pages around important topics.

Instead of random pages, SideGuy builds topic clusters.

Clusters help search engines understand authority.

--------------------------------------------------

EXAMPLE CLUSTER

Topic:

software development cost

Pages:

software-development-cost-for-startups  
software-development-cost-for-small-business  
software-development-cost-per-hour  
software-development-cost-breakdown  
software-development-cost-for-saas  

--------------------------------------------------

RULES

append-only  
use seo-template.html  
connect pages to knowledge hubs  
add pages to sitemap.xml  
add pages to index.html  

--------------------------------------------------

GOAL

Build deep topical authority clusters across the platform.
