# SideGuy Auto Discovery Engine

Purpose:
Turn SideGuy into a living system that discovers new
problem topics and technology signals automatically.

Workflow:

1. ingest signals
2. normalize idea
3. classify theme
4. generate reserve topics
5. score opportunity
6. queue page candidates
7. queue upgrade candidates
8. log actions

Principles:

append-only
reserve-first
controlled publishing
human review

SideGuy expands by discovering real signals
instead of randomly generating content.
