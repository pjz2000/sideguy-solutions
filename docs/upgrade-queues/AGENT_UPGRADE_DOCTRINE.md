# Agent Upgrade Doctrine

## Goal

Upgrade the highest-opportunity existing SideGuy pages first.

## Priority order

1. Pages with weak human CTA
2. Pages with weak internal links
3. Pages missing FAQ
4. Pages missing freshness / timestamp
5. Thin pages with traffic or money intent
6. Important hubs that should distribute authority

## Required upgrade blocks

- Text PJ orb / visible human CTA
- FAQ block
- process / how it works block
- 5 to 10 contextual internal links
- local trust language where relevant
- timestamp / build version / last updated
- structured data where appropriate
- one practical example or scenario

## Positioning

Every page should increasingly reinforce:

- clarity before cost
- AI for explanation, human for decision
- calm local operator help
- SideGuy as the trusted human layer
