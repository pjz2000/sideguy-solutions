# Million Page Reserve — Phase 2 Notes

Phase 2 adds operational control:

- state packs
- reserve scoring
- anti-duplicate filtering
- controlled wave publishing
- category sitemap partitioning
- sitemap index generation
- homepage link append
- wave logging

Why this matters:
At a million-page target, the job is not "publish everything."
The job is:
- build reserve space
- score intelligently
- publish controlled winners
- keep sitemap architecture clean
- prevent low-quality duplication

This is the transition from:
"huge idea"
to
"operator system"
