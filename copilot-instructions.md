# SideGuy Copilot Operating Rules

## Automation Allowed

- Analysis and reporting (read-only tools)
- Cluster planning and page briefs
- Draft pages (for human review before publishing)
- Page enhancement on existing pages (FAQ schema, calculators, comparison tables, internal links)

## Automation NOT Allowed

- Auto-publishing or auto-committing generated content to main
- Bulk template page generation (`{city}` + `{service}` variable substitution)
- Scheduled cron systems that generate or publish content

## Publishing Rule

All content requires human review before it goes live.  
Automation assists every step of the workflow **except the final publish decision**.
